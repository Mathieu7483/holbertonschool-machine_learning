## 📊 Description

Ce projet est une introduction complète à la visualisation de données en Python. À l'aide de la bibliothèque **Matplotlib**, ce module explore comment transformer des données numériques (provenant souvent de NumPy) en graphiques clairs et informatifs. La maîtrise du plotting est essentielle pour l'analyse de données et le Machine Learning afin de comprendre la distribution des données et les résultats des modèles.

## 🎓 Objectifs d'apprentissage

À la fin de ce projet, je suis capable d'expliquer :

  * **Types de graphiques** : Quand utiliser un Scatter plot, un Line graph, un Bar graph ou un Histogramme.
  * **Matplotlib** : Savoir naviguer dans l'API `pyplot`.
  * **Personnalisation** : Ajouter des labels, des titres, des légendes et changer les échelles (logarithmique, etc.).
  * **Multi-visualisation** : Créer des subplots et superposer plusieurs jeux de données sur un même graphique.
  * **3D Plotting** : Visualiser des données complexes dans un espace tridimensionnel.

## 🔬 Focus Technique : Task 8 (PCA)

La **PCA (Principal Component Analysis)** est utilisée ici pour réduire les 4 caractéristiques des fleurs d'Iris (longueur/largeur des sépales et pétales) en 3 dimensions principales (U1, U2, U3). Cela nous permet de visualiser dans un cube 3D comment les différentes espèces (Setosa, Versicolor, Virginica) se regroupent naturellement.
