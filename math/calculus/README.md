# Calculus for Machine Learning

## 📈 Description

Ce projet pose les bases mathématiques nécessaires à la compréhension des algorithmes d'optimisation en Intelligence Artificielle. Il couvre les notations de sommation et de produit, les règles de dérivation (essentielles pour la *Backpropagation*) et les principes de l'intégration. L'objectif est de passer de la théorie mathématique à son application logique dans le développement de systèmes intelligents.

## 🎓 Objectifs d'apprentissage

À la fin de ce projet, je suis capable d'expliquer :

  * **Notations Sigma ($\sum$) et Pi ($\prod$)** : Manipulation des sommes et des produits mathématiques.
  * **Dérivées** : Compréhension de la pente, de la règle du produit et de la règle de la chaîne (*Chain Rule*).
  * **Dérivées partielles** : Fondement du gradient descendant dans les espaces multidimensionnels.
  * **Intégrales** : Calcul d'aires sous la courbe, intégrales définies et indéfinies.
  * **Séries** : Compréhension des suites et séries mathématiques communes.

## 🛠️ Configuration et Spécifications

  * **OS** : Ubuntu 20.04 LTS.
  * **Langage** : Python 3.9.
  * **Style de code** : Respect de `pycodestyle` (version 2.11.1).
  * **Format** : Mélange de scripts Python et de fichiers de réponses textuelles pour les questions à choix multiples.
